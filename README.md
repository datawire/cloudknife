# Cloudknife

The `datawire/cloudknife` is an alpine container with many useful tools already installed:

- Python 3
- Python `redis` library
- `dig`
- `nslookup`
- `curl`
- `wget`
- `netcat`
- `socat`
- `pgcli` (PostgreSQL client)
- `gcloud` (Google Cloud Tools)
- `cloud_sql_proxy` (Google Cloud SQL)
